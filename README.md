# Caesar-Cipher

 Basic Caesar Cipher encryptor written in JavaScript

# How it Works

- The program takes a phrase and a shift number as inputs from process.argv.
- The phrase is encrypted by shifting each letter, based on its position in the english alphabet, by the specified number.
- Case insensitive
- A negative shift means shift to the left
- A positive shift means shift to the right
- The encrypted phrase is output to the console.
